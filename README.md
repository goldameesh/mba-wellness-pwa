# BAS MBA Wellness PWA

Ready-to-upload GitHub Pages package for the MBA Wellness booklet.

## Included
- `index.html` – main app
- `manifest.webmanifest` – install metadata
- `sw.js` – offline caching
- `icons/` – BAS-branded home screen icons
- `previews/` – preview/screenshot assets for app listing and sharing

## Publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this folder to the repository root.
3. In GitHub: **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select **main** branch and **/(root)** folder.
6. Save. GitHub Pages will publish the site.

For a custom repo name, the package already uses relative paths so it should work on standard GitHub Pages project URLs.
